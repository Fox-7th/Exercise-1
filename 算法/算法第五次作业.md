$li$代表两数比较后产生的较大值，$si$代表两数比较后产生的较小值

用6次比较在5个元素中找中位数：
    
    Algorithm FindMid_In6(A[])
    （l1,s1）<- Compare(A[0],A[1]);
    （l2,s2）<- Compare(A[0],A[1]);
    
    l3 <- Min(l1,l2);
    s4 <- Max(s1,s2);
    
    s5 <- Min(l3,A[2]);
    mid <-Max(s4,s5);
    
    return mid;
